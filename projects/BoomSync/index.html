BoomSync

Designer, April 2017

PLAY GAME

BoomSync is a short online game that teaches the basics of Asynchronous programming in JavaScript.
The project won first place at Dartmouth's 12-hour hackathon, Hackday II.
Visit the repo here.

We decided to create an educational game, inspired by existing games such as Flexbox Froggy and CSS Diner.
All assets were created in Adobe Illustrator and saved as SVGs.
